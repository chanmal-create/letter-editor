文字配置エディタ
note のアイキャッチ画像などを作るための、文字を1文字ずつ自由に配置できるツールです。
使い方
	1.	index.html をブラウザで開く（ダブルクリックでOK）
	2.	テキストを入力すると、1文字ずつ操作できるオブジェクトになる
	3.	文字をドラッグで移動、ハンドルで拡大縮小・回転
	4.	ダブルクリックでその文字だけ縦書き／横書きを切替
	5.	Shift+クリック、または空白からのドラッグで複数選択 → まとめて縦横切替・サイズ揃え・間隔揃え
	6.	「PNGを書き出す」で画像として保存
GitHub Pages で公開する場合
このリポジトリの Settings → Pages で公開元ブランチを指定するだけで、
index.html がそのままトップページとして表示されます。
技術
	•	完全に単一HTMLファイル（外部サーバー不要）
	•	フォントは Google Fonts を CDN 経由で読み込み
	•	依存ライブラリなし（素のJavaScript / Canvas API）
